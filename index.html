Endpoints Used

GET https://api.openweathermap.org/data/2.5/weather GET https://api.openweathermap.org/data/2.5/forecast

Parameters

  • q: City name (e.g., "London")
  • appid: API Key for authentication
  • units: metric (for Celsius)
